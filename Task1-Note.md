### 2、运算符

**算数运算符**

```python
# + 加
print(1 + 2) # result = 3

# - 减
print(8 - 1) # result = 7

# * 乘
print(2 * 3) # result = 6

# / 除
print(3 / 4) # result = 0.75

# // 整除
print(3 // 4) # result = 0

# % 取余
print(3 % 4) # result = 3

# ** 幂
print(2 ** 3) # result = 8

```
